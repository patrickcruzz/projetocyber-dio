
# Projeto Cyber DIO — Kali Linux + Medusa

## Objetivo
Criar um laboratório controlado para testes de força bruta e enumeração utilizando Kali Linux e Metasploitable2.

## Ferramentas Utilizadas

- Kali Linux
- Medusa
- Metasploitable2
- DVWA
- VirtualBox
- Nmap
- GitHub

## Ambiente

Foi criado um ambiente virtualizado com duas máquinas:

- Kali Linux (máquina atacante)
- Metasploitable2 (máquina vulnerável)

## Etapas Realizadas

1. Configuração das VMs
2. Configuração de rede host-only
3. Descoberta de IP
4. Teste de conectividade
5. Enumeração de serviços
6. Testes com Medusa
7. Password Spraying SMB
8. Documentação dos resultados
